How Storganize works
An app that will make organizing storage easier and more efficient.

Storganize is an app that allows you to create better organization for your storage bins and moving boxes. This allows you to orgnaize much easier. The app allows you to create boxes which allow you to give it a title, description and tags (ie. Holidays, Seasons). You then can add items to the box, once you hit create a qr code will be generated and you can print it out and stick it on the box. You can scan the QR code and it will provide a link to your storganize box.
